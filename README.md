## Team Members
- Lucas Zhang

## Dataset
Water Potability dataset from Kaggle

## Project Structure
- `data/` contains the dataset
- `src/` contains all code files
- `results/` contains outputs and plots

## How to Run
1. Open terminal in the project folder
2. Go to the source folder:
   `cd src`
3. Run:
   `python main.py`

## Models Compared
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
